Halo Aku Ahmad Dari Universitas Sumatera Utara. ini adalah Tempat belajar dasar pengembangan web dengan cara menyenangkan mulai dari HTML, CSS, JavaScript hingga Tailwind CSS.



Apa Itu Web?

Web (World Wide Web) adalah sistem yang memungkinkan kita mengakses dan menampilkan informasi di internet menggunakan browser seperti Chrome atau Firefox.

Website dibangun dengan HTML untuk struktur, CSS untuk tampilan, dan JavaScript untuk interaksi. Semua bekerja sama agar pengguna mendapatkan pengalaman yang menarik dan interaktif.

Kini, web telah menjadi bagian penting kehidupan kita .mulai dari belajar, hiburan, hingga bekerja, semuanya bisa dilakukan lewat dunia maya.
